ตั้งค่า API URL

ทำครั้งเดียว — ระบบจะจำให้อัตโนมัติ

📋 วิธีรับ URL จาก Google Apps Script

1
เปิด Google Sheet → เมนู Extensions → Apps Script
2
ลบโค้ดเดิมออก วางโค้ดใหม่ที่ได้รับ กด Save (💾)
3
กด Deploy → New deployment → เลือก Web app
ตั้ง Execute as: Me  |  Who has access: Anyone
4
คลิก Deploy → คัดลอก Web app URL มาวางด้านล่าง

⚠️ URL ที่ถูกต้องจะมีหน้าตาแบบนี้:

https://script.google.com/macros/s/XXXXX/exec
